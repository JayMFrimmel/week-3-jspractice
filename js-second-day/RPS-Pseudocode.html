Rock Paper Scissors Pseudocode

1. User chooses R, P, or S on keyboard representing Rock, Paper or Scissors

2. Computer chooses a random value in a list of R. P, or S.

We determine if the user won or not.

  • If user picked rock and computer picked scissors then user wins.

  • If user picked rock and computer picked paper then user loses.

  • If user picked scissors and computer picked rock then user loses.

  • If user picked scissors and computer picked paper user wins.

  • If user picked paper and computer picked rock then user wins.

  • If user picked paper and computer picks scissors the user loses.

4. We then add to their score.

  • If user won then we would add one to their wins.

    If user lost then we would add one to their losses.

    If the user ties then we would add one to their ties.

5. The user then presses: R, P, or S to play again.
